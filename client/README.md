# Client (Web UI)

This is the React + Tailwind CSS web UI for the Realtime Audio and Video Capturing and Streaming System.

## Features
- Live stream viewing
- Audio/video recording
- Playback of stored recordings

## Setup
1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm start
   ```

## Notes
- Requires the backend server to be running for API and signaling.
